## Hogwarts-escape-room

The Hogwarts Escape Room Game is an interactive, Harry Potter-themed game that takes players on a journey through the wizarding world of Harry Potter.

### Technologies-used

1. Front-end
   - HTML
   - CSS/BOOTSTRAP
   - JavaScript/jQuery
2. Back-end
   - DJango (for database management)
